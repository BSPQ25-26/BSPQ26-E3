Detailed Description

Service layer for shopping-cart operations.

This class exposes the remote business interface for managing a user's shopping cart. It allows clients to add/remove items, clear the cart and perform a full checkout (payment + stock update + receipt generation).

All write operations are executed inside Spring transactions so the cart never ends up in an inconsistent state.
