Detailed Description

Service layer that simulates the order lifecycle.

Manages state transitions for receipts (PROCESSING → DELIVERY → COMPLETED) and supports cancellation while the order is still in PROCESSING. Durations are deliberately short (seconds) so that the workflow can be observed during a demo.
